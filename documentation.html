Dokumentacja

1. Pobranie plików

Pobierz zipa. Pózniej trzeba go będzie rozpoakować i wgrac na serwer.
Znajdują się tam libki które umożliwiają uruchomienie aplikacji na systemie Linux, gdzie wersja Node to 12.
Poniżej przedstawimy możliwą instalację, korzystając z Heroku (serwera, który udostępnia nam dużo możliwości i przy pomocy Atlas Mongo(bazy danych potrzebnej do działania aplikacji)

2. Utworzenie darmowej bazy MongoDb przy pomocy

  • Tworzymy konto wchodząc na link: https://account.mongodb.com/account/register
  • Po utworzeniu konta, tworzymy darmowy klaster. Strona powinna nas pokierowac jak to zrobić
  • Czekamy, aż stworzy się klaster
Najpier możemy założyć sobie bazę danych Mongo. Możemy to zrobić wchodząc na atlas.mongo.pl

2. Utworzenie darmowego konta na Heroku

  • Tworzymy konto wchodząc na link: https://account.mongodb.com/account/register
  • Po utworzeniu konta, tworzymy darmowy klaster. Strona powinna nas pokierowac jak to zrobić
  • Czekamy, aż stworzy się klaster
Następnie zakładamy konto na heroku, tworzymy sobie nową stronę.
Ustawiamy sobie Reveal Config: takie jak Path, Node_env, Mongo_URI pobrane z wcześniej utworzonej bazy.
Ustawiamy buildPacka
Pobieramy heroku CLI zgodnie z instrukcją:
Kiedy mamy już folder z naszą aplikacja, tam rozpakowujemy zipa, tak aby w głównym folderze był plik Procfile.
Wgrywamy tak jak napisano w instrukcji heroku: git add. git commit i git push heroku master
Ważne żeby przed tym upewnić się że ustawiliśmy BuildPacka i Reveal Config.
